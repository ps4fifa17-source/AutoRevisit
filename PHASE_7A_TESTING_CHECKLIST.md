# Phase 7A Testing Checklist

## Starter test

Set dealer package to starter.

Check:

- Create page only shows Simple customer goal.
- Reg lookup button is locked.
- Create more than 10 live pages is blocked.
- Customer pages show Powered by AutoRevisit.
- Leads/Insights show locked Pro messaging.

## Professional test

Set dealer package to pro.

Check:

- Simple, Finance, Family, Performance visible.
- Premium and Executive hidden/locked.
- Reg lookup works.
- 50 page limit.
- Lead scoring visible.
- Analytics visible.
- Watermark still shows unless changed later.

## Premium test

Set dealer package to premium.

Check:

- All customer goals visible.
- Unlimited pages.
- Premium and Executive customer pages work.
- Watermark can be removed.
- Full analytics visible.

## Admin test

Set your profile role to admin.

Check:

- Admin appears in sidebar.
- Admin page opens.
- Dealer plan changes save to Supabase.
- Package tester changes your test plan.
- Clearing test plan returns to live dealer plan.

## Customer page test

Open customer page on phone.

Check:

- Dealer logo appears if set.
- Dealer colour controls buttons and accents.
- Style/goal layouts look different.
- WhatsApp click records event.
- Call click records event.
- Finance/reserve clicks record event.
